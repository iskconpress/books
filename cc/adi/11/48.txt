====== CC Ādi 11.48 ======

===== Text =====

> nakaḍi, mukunda, sūrya, mādhava, śrīdhara
> rāmānanda vasu, jagannātha, mahīdhara

===== Synonyms =====

[[synonyms:n:nakaḍi]]---of the name Nakaḍi; [[synonyms:m:mukunda]]---of the name Mukunda; [[synonyms:s:sūrya]]---of the name Sūrya; [[synonyms:m:mādhava]]---of the name Mādhava; [[synonyms:ś:śrīdhara]]---of the name Śrīdhara; [[synonyms:r:rāmānandavasu]]---of the name Rāmānanda Vasu; [[synonyms:j:jagannātha]]---of the name Jagannātha; [[synonyms:m:mahīdhara]]---of the name Mahīdhara.

===== Translation =====

**Among Lord Nityānanda's devotees, Nakaḍi was the thirty-seventh, Mukunda the thirty-eighth, Sūrya the thirty-ninth, Mādhava the fortieth, Śrīdhara the forty-first, Rāmānanda the forty-second, Jagannātha the forty-third and Mahīdhara the forty-fourth.**

===== Purport =====

Śrīdhara was the twelfth //gopāla.//

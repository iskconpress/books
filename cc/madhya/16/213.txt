====== CC Madhya 16.213 ======

===== Text =====

> ataeva ihāṅ tāra nā kailuṅ vistāra
> punarukti haya, grantha bāḍaye apāra

===== Synonyms =====

[[synonyms:a:ataeva]]---therefore; [[synonyms:i:ihāṅ]]---here; [[synonyms:t:tāra]]---of that incident; [[synonyms:n:nā]] [[synonyms:k:kailuṅ]]---I did not give; [[synonyms:v:vistāra]]---elaboration; [[synonyms:p:punarukti]]---repetition; [[synonyms:h:haya]]---it is; [[synonyms:g:grantha]]---the book; [[synonyms:b:bāḍaye]]---increases; [[synonyms:a:apāra]]---unlimitedly.

===== Translation =====

**I will not narrate these incidents because they have already been described by Vṛndāvana dāsa Ṭhākura. There is no need to repeat the same information, for such repetition would unlimitedly increase the size of this book.**

====== Cc. Madhya 21.67 ======

===== Text =====

> daśa-biśa-śata-sahasra-ayuta-lakṣa-vadana
> koṭy-arbuda mukha kāro, nā yāya gaṇana

===== Synonyms =====

[[synonyms:d:daśa]]---ten; [[synonyms:b:biśa]]---twenty; [[synonyms:ś:śata]]---hundred; [[synonyms:s:sahasra]]---thousand; [[synonyms:a:ayuta]]---ten thousand; [[synonyms:l:lakṣa]]---a hundred thousand; [[synonyms:v:vadana]]---faces; [[synonyms:k:koṭi]]---ten million; [[synonyms:a:arbuda]]---a hundred million; [[synonyms:m:mukha]]---faces; [[synonyms:k:kāro]]---of some of them; [[synonyms:n:nā]] [[synonyms:y:yāya]] [[synonyms:g:gaṇana]]---not possible to count.

===== Translation =====

**"These Brahmās had different numbers of heads. Some had ten heads, some twenty, some a hundred, some a thousand, some ten thousand, some a hundred thousand, some ten million and others a hundred million. No one can count the number of faces they had.**
